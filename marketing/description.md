## Description (≤4000 chars)

```
MachStruct is a fast, native macOS viewer and editor for structured data —
JSON, XML, YAML, and CSV. Built for developers, data scientists, and anyone
who lives inside API payloads, exports, and config files.

WHY MACHSTRUCT

• Designed for huge files. 100 MB JSON loads in under half a second on Apple
  Silicon. Memory-mapped reads and lazy node materialisation mean MachStruct
  stays responsive where browser-based tools and Electron alternatives stall.

• Native macOS, all the way down. SwiftUI, Quick Look, Spotlight, Services,
  drag-and-drop, dark mode, hardware acceleration — everything you'd expect
  from a polished Mac app.

• Format-agnostic. The same tree, table, and raw views work across JSON,
  XML, YAML, and CSV. Convert between them in a click.

CORE FEATURES

• Tree view with single-click expand, drag-and-drop reordering, bookmarks,
  and inline editing.
• Table view for CSV with per-column statistics — counts, ranges, unique
  values.
• Raw view with syntax highlighting, pretty-print and minify modes, and
  search.
• Full-text search across keys and values, with auto-expansion of matched
  ancestors and prev/next navigation.
• Bookmarks survive close and reopen — pinned to file path, not session.
• Command palette (⇧⌘P) — every menu, toolbar action, recent file, and
  bookmark, fuzzy-searchable.
• Scalar inspector — detects base64, Unix timestamps, ISO 8601 datetimes,
  UUIDs, and hex colours inside string values, and shows them decoded.
• Auto-fixer for malformed JSON — trailing commas, single quotes, unquoted
  keys, JS-style comments. One click and you're parsing again.
• Quick Look preview extension — peek at structured files from Finder
  without opening the app.
• Spotlight importer — search by content, not just filename.
• Services menu — "Format with MachStruct" or "Minify with MachStruct" any
  selected text in any app.
• Clipboard watch — auto-detects structured data on the clipboard and
  offers to open it.

PRIVACY

MachStruct is a local-first, single-purpose tool. It runs in the macOS
sandbox, only reads files you explicitly open or drop on the window, and
makes no network requests beyond optional update checks (which are
disabled in the App Store version). No analytics, no telemetry, no account.

LOCALISATION

Available in English and German.

REQUIREMENTS

macOS 14.0 (Sonoma) or later. Universal binary — Apple Silicon and Intel.
```
